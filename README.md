# 📊 Algorithm Visualizer

Interactive Algorithm Visualizer built using HTML, CSS, and JavaScript to help users understand Data Structures and Algorithms through real-time animations.

---

## 🚀 Features

* 📈 Sorting Algorithm Visualization
* ⚡ Real-Time Animation
* 🎨 Interactive UI
* 🧠 Educational Learning Tool
* 📱 Responsive Design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```bash
Algorithm-Visualizer/
│
├── algorithms/
├── utils/
├── index.html
├── style.css
├── script.js
└── .gitignore
```

### Clone Repository

```bash
git clone https://github.com/harshdave05/Algorithm-Visualizer.git

```

### Run Project

Simply open:

```bash
index.html
```

in your browser.
